# Notice

`tinycpu-pynq` is an independent clean-room educational FPGA project.

The RTL, simulation tests, programs, Vivado scripts, and documentation in this
repository are written for this repository and are intended to be reproducible
from a public GitHub clone.

This project is not a course homework solution and does not depend on private
or local course directories. It targets standard RISC-V RV32IM and implements a
small AXI-Lite SoC from first principles for learning purposes.
