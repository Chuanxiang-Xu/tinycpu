# Agent Notes

This repository is source-first. Keep generated FPGA and simulation artifacts
out of version control.

Guidelines:

- Keep RTL clean-room and educational.
- Do not copy or depend on private course solution code.
- Do not reference local/private coursework directories in source or docs.
- Prefer simple SystemVerilog over clever abstractions.
- Keep Vivado projects reproducible from Tcl.
- Keep cocotb tests runnable from a fresh clone with `requirements.txt`.
